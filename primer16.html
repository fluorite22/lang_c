Ряд Тейлора для sin(x):
sin(x) = x - x³/3! + x⁵/5! - x⁷/7! + ... + (-1)ⁿ·x²ⁿ⁺¹/(2n+1)!
sin(x) ≈ x - x³/6 + x⁵/120 - x⁷/5040 + ...

0.841471

Пошаговый процесс вычисления
s - сумма ряда
a - текущий элемент
xx = xⁿ
fact = n!
z - знак (+1/-1)
n xx (xⁿ) fact (n!) z (знак) a (элемент) s (сумма) |a|
Всего итераций: 0
double sin1 ( double x )
{
    double s = 0, a, xx = x, fact = 1, n2 = 1;
    int z = 1;
    do {
        a = z * xx / fact;
        s += a;
        z = -z;
        xx *= x*x;
        n2 += 2;
        fact *= (n2-1) * n2;
    } while ( fabs(a) > 1.e-6 );
    return s;
}
Обозначения в коде:
• s - частичная сумма ряда
• a - элемент ряда (-1)ⁿ·x²ⁿ⁺¹/(2n+1)!
• xx - значение x²ⁿ⁺¹
• fact - значение (2n+1)!
• n2 - значение 2n+1
• z - знак (+1 или -1)
